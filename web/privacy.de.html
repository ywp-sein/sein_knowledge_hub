Datenverarbeitung durch diese Website

Diese Website ist statisch. Sie enthält kein Kontaktformular, Newsletterformular, Analyseskript, Werbetracker oder eigenes Cookie-Banner. Die Darstellungseinstellung wird nur im Browser der Besucherin oder des Besuchers gespeichert und nicht an die Betreiberin oder den Betreiber übertragen.

Die Website wird über GitHub Pages gehostet. GitHub gibt an, dass beim Besuch einer GitHub-Pages-Website die IP-Adresse der Besucherin oder des Besuchers zu Sicherheitszwecken protokolliert und gespeichert wird, unabhängig davon, ob die Person bei GitHub angemeldet ist.1

Externe Links

Der Knowledge Hub verlinkt auf externe Websites wie Behörden, soziale Organisationen, Forschungsquellen und GitHub. Diese externen Websites sind für ihre eigenen Datenschutzpraktiken verantwortlich.
