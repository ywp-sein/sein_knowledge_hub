Data processed by this website

This website is static. It does not include a contact form, newsletter form, analytics script, advertising tracker, or custom cookie banner. The theme setting is stored only in the visitor's browser through local storage and is not transmitted to the site maintainer.

The website is hosted on GitHub Pages. GitHub states that when a GitHub Pages site is visited, the visitor's IP address is logged and stored for security purposes, whether or not the visitor is signed into GitHub.1

The legal basis for technically necessary access data is the legitimate interest in delivering a secure and reliable website. The site maintainer does not set an independent storage period for GitHub Pages server logs.

Email contact

If you contact the maintainer by email, the personal data contained in that message will be processed to read, answer, and document the request. Emails are kept only as long as needed for the conversation, legal obligations, or legitimate documentation interests.

The Knowledge Hub links to external websites such as public authorities, social organizations, research sources, and GitHub. Those external websites are responsible for their own privacy practices.
