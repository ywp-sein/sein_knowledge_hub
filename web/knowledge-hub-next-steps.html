Possible improvements

  • Add a formal source review field to each article: checked by, checked date, and next review date.
  • Expand the organization list with categories such as emergency, medical, food, hygiene, counselling, and volunteering.
  • Create a printable one-page homelessness quick guide in English and German.
  • Add a submission workflow for new sources and corrections.
  • Add a page template for future issue categories.
  • Review accessibility: keyboard navigation, contrast, table behavior on mobile, and language attributes.

Update practice

When an improvement is completed, add a row to the version log and update the revision timestamp on every changed page. Larger changes should also include a short note about which sources were checked.
