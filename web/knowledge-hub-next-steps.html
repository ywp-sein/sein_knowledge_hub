Possible improvements

  • Add a reusable sidebar partial or build step so page navigation does not need to be edited by hand on every page.
  • Add a formal source review field to each article: checked by, checked date, and next review date.
  • Expand the organization list with categories such as emergency, medical, food, hygiene, counselling, and volunteering.
  • Create a printable one-page homelessness quick guide in English and German.
  • Add a submission workflow for new sources and corrections.
  • Add a page template for future issue categories.
  • Review accessibility: keyboard navigation, contrast, table behavior on mobile, and language attributes.

GitHub issue label set

Use source:* for facts, citations, links, organizations, services, and resource listings. Use content:* only for wording and article structure, so people filing issues do not need to choose between overlapping labels.

Label Color Use when
content: wording clarity #d73a4a Improve phrasing, readability, explanation, or tone without changing factual meaning.
content: structure #d73a4a Reorganize headings, sections, summaries, tables, or page flow without adding external facts.
source: needed #fbca04 A claim, number, quote, policy statement, service detail, or directory entry needs a source.
source: verify #fbca04 An existing citation, organization, service, phone line, link, or directory listing needs checking.
source: add #fbca04 A useful external source should be added, such as an official page, report, dataset, law, organization, service, or help offer.
source: update #fbca04 An existing source or listing has changed details such as date, location, eligibility, contact, availability, or scope.
source: remove #fbca04 A source or listing is broken, duplicated, outdated, closed, unreliable, irrelevant, or should not be recommended.
translation: missing #1d76db Add German or English text where one language version is incomplete or absent.
translation: mismatch #1d76db Fix places where English and German pages do not contain the same meaning or detail.
translation: wording #1d76db Improve German or English translation style while preserving the same content.

Update practice

When an improvement is completed, add a row to the version log and update the revision timestamp on every changed page. Larger changes should also include a short note about which sources were checked.
